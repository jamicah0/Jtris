# JTRIS
Tetris made in Java. This was a project made for a school assignment.

## Features
- Fully playable Tetris game in Java
- Marathon mode
  - Tetris with modern guidelines
- Classic mode
  - Tetris based on the original NES version

## Controls
- Left Arrow: Move piece left
- Right Arrow: Move piece right
- Up Arrow: Rotate piece clockwise
- Down Arrow: Soft drop
- CTRL: Rotate piece counter-clockwise
- Spacebar: Hard drop
- ESC: Pause/Resume game

## How to Run
1. Make sure you have JVM installed on your machine
2. Download the JTRIS.jar file from the releases section
3. Open JTRIS.jar
